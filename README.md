# 🔎오구생활 - 학교 생활에 도움이 되는 앱 개발 프로젝트

### 개요
* 학교 생활을 하는데 도움이 되는 다양한 기능을 가진 안드로이드 애플리케이션
  * 오구라는 귀여운 캐릭터를 학생들의 모습에 빗대어 UI 설계
  *학식 추천, 맛집 추천, 메모장, 계산기 등의 기능 포함
